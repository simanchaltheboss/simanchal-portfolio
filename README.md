## ⚡ About The Project
This repository is the production source code for the personal and business portfolio of Simanchal Bordoloi. It is engineered from the ground up to solve the most pressing issue in modern web development: **bloat**. 

By ignoring heavy, unneeded frontend frameworks and relying on tightly optimized vanilla JavaScript and CSS, this application guarantees instant page loads, maximizing B2B client conversions and SEO crawlability.

## 🛠️ Architecture & Tech Stack
- **DOM Structure:** Semantic HTML5 strictly conforming to WCAG accessibility guidelines.
- **Styling:** Modular Vanilla CSS utilizing CSS Variables (`--primary`, `--bg-main`), flexbox grid fallbacks, and mathematically calculated clamping for fluid typography.
- **Interactivity:** Vanilla ES6+ JavaScript. No jQuery. No React. 0 dependencies.
- **Theming:** Persistent Light/Dark theme driven by `localStorage` states.

## ✨ Core Features
1. **Dynamic Theming System:** Instant transitioning between a minimalist Light Mode and a deep, glassmorphic Dark Mode interface.
2. **Micro-Interaction Polish:** High-end hover transitions on project cards (soft zooms, lifts, and shadow expansion) to convey premium SaaS aesthetics.
3. **Conversational UX:** Floating Sticky WhatsApp CTA integration bypassing traditional sterile contact forms, converting leads directly to active communication channels.

## 🗺️ Local SEO Implementation
The codebase possesses hardcoded edge optimizations aimed specifically at the Assam geographic tech market. 
* Implementation of `ProfessionalService` JSON-LD structured data mapping.
* Hard-coded Geo-targeting for **Guwahati, Tezpur, Nagaon, and Dhekiajuli**.
* Semantic hierarchy targeting WooCommerce and Custom Service keywords.

## 📊 Analytics
This project uses Vercel Web Analytics for privacy-focused visitor tracking. The analytics script is loaded via CDN and requires no configuration. Once deployed to Vercel, analytics will automatically start tracking:
- Page views
- Unique visitors
- Referrer information
- Device and browser data

Analytics data is available in your Vercel dashboard after enabling Web Analytics for your project.

## 🚀 Getting Started

To get a local copy up and running, no Node.js package managers or build procedures are required. 

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- *Optional:* Visual Studio Code equipped with the "Live Server" extension for hot reloading.

### Installation
1. Clone the repository
   ```sh
   git clone https://github.com/simanchaltheboss/simanchal-portfolio.git
   ```
2. Navigate into the directory
   ```sh
   cd simanchal-portfolio
   ```
3. Open `index.html` in your browser.
